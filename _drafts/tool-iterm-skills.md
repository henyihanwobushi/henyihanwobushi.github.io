| Settings          | Description                                                |
| :---------------- | :--------------------------------------------------------- |
| `⌘ + D`           | Split pane vertically                                      |
| `⌘ + ⇧ + D`       | Split pane horizontally                                    |
| global hotkey     | `⌘ + ,` -> Keys -> Hotkey                                  |
| use last tabs dir | `⌘ + ,` -> Profiles -> General -> Reuse last session's dir |
