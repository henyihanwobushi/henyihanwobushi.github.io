VSCode is a great editor. It's free, fast, and extensible. You can use it to edit almost any kind of files. It's a great tool for developers.

## Keymaps

| Function                 | Keymap          |
| :----------------------- | :-------------- |
| Open file                | `⌘ + P`         |
| Create new file          | `⌘ + N`         |
| Save file                | `⌘ + S`         |
| Close file               | `⌘ + W`         |
| Close all files          | `⌘ + K` `⌘ + W` |
| Partial Accept Promption | `⌘ + →`         |
